Usage:
  grog [command]

Available Commands:
  build       Loads the build configuration and executes targets
  clean       Removes build outputs and clears the cache
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command

Flags:
      --color string   Set color output (y1) (default "auto")
  -h, --help           help for grog

Use "grog [command] --help" for more information about a command.
